Mysql table import and export 快速导出导入

2019-12-20 大熊BIGBEAR►Mysql Mysql, 大熊BIGBEAR 评论 字数统计: 132(字) 阅读时长: 1(分)

导出

  • –quick，-q
    该选项在导出大表时很有用，它强制 mysqldump 从服务器查询取得记录直接输出而不是取得所有记录后将它们缓存到内存中。

  • –extended-insert, -e
    使用具有多个VALUES列的INSERT语法。这样使导出文件更小，并加速导入时的速度。默认为打开状态，使用–skip-extended-insert取消选项。

  • -t
    仅导出表数据，不导出表结构
